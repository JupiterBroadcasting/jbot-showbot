# Showbot

A sweet IRC bot for [5by5](http://5by5.tv). Built on [cinch](https://github.com/ymendel/cinch/blob/master/lib/cinch/base.rb).

## Commands

* show 
<pre>
!show b2w 12
showbot> http://5by5.tv/b2w/12
</pre>
